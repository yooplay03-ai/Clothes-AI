# 🧥 MoodFit — 기분·날씨·상황 기반 AI 코디 추천 서비스

> "오늘 뭐 입지?" 를 없애주는 나만의 스타일 어시스턴트

---

## 1. 프로젝트 개요

### 1-1. 서비스 소개

MoodFit은 사용자의 **기분, 날씨, 만나는 대상, 일정** 등 상황 정보를 입력받아 AI가 최적의 코디를 추천해주는 서비스입니다. 내 옷장에 등록된 실제 옷을 기반으로 추천하며, 커뮤니티를 통해 다른 사용자의 코디를 참고할 수도 있습니다.

### 1-2. 목표

- 매일 아침 옷 고르는 시간과 스트레스를 줄여준다
- 사용자의 실제 보유 의류 기반으로 현실적인 코디를 제안한다
- 날씨·상황에 맞는 스마트한 추천으로 외출 만족도를 높인다
- 커뮤니티를 통해 스타일 영감을 주고받을 수 있다

### 1-3. 타겟 사용자

| 구분 | 설명 |
|------|------|
| 1차 타겟 | 20~30대 직장인 — 매일 출근 코디가 고민인 사람 |
| 2차 타겟 | 패션에 관심 있는 10~20대 — 다양한 스타일을 시도하고 싶은 사람 |
| 3차 타겟 | 옷은 많지만 조합이 어려운 모든 연령대 |

### 1-4. 플랫폼

- **웹 앱** (React / Next.js 기반 반응형)
- **모바일 앱** (React Native 또는 Flutter 크로스 플랫폼)
- 웹과 앱 간 데이터 완전 동기화

## 2. 핵심 기능 상세

### 2-1. 🤖 AI 기반 코디 추천

사용자가 입력한 상황 정보를 종합하여 AI가 코디를 생성합니다.

**입력 정보 (추천 요청 시)**

| 항목 | 입력 방식 | 예시 |
|------|-----------|------|
| 기분 | 이모지/슬라이더 선택 | 😊 신남, 😐 무난하게, 💼 진지하게 |
| 만나는 대상 | 태그 선택 | 직장 동료, 소개팅, 친구, 가족, 면접 |
| 일정/장소 | 태그 선택 + 텍스트 | 카페, 회의, 야외활동, 결혼식 |
| 스타일 선호 | 선택 (선택사항) | 캐주얼, 포멀, 미니멀, 스트릿 |
| 날씨 | 자동 (위치 기반) | 기온, 강수확률, 자외선 지수 |

**추천 결과 출력**

- 내 옷장 기반 코디 2~3세트 카드형 제시
- 각 코디에 대한 AI의 추천 이유 한 줄 설명
- 코디 전체 미리보기 (옷장 사진 조합 또는 일러스트)
- "이 코디로 결정" 버튼 → 오늘의 코디 기록 저장

**AI 추천 로직 핵심 요소**

```
추천 점수 = 날씨 적합도(30%)
           + 상황 적합도(25%)
           + 기분 매칭(20%)
           + 색상 조화(15%)
           + 최근 착용 빈도 역가중치(10%)
```

### 2-2. 👕 내 옷장 등록/관리

사용자가 보유한 의류를 등록하고 관리하는 기능입니다.

**의류 등록 방법**

1. **사진 촬영/업로드** → AI가 자동으로 카테고리·색상·스타일 태그 분류
2. **수동 입력** → 카테고리, 색상, 브랜드, 계절감 등 직접 입력
3. **쇼핑몰 연동** (향후) → 구매 내역에서 자동 불러오기

**의류 정보 구조**

| 필드 | 설명 | 입력 방식 |
|------|------|-----------|
| 사진 | 의류 사진 1~3장 | 촬영/갤러리 |
| 카테고리 | 상의/하의/아우터/신발/액세서리 | AI 자동 + 수정 |
| 서브 카테고리 | 티셔츠, 셔츠, 청바지, 슬랙스 등 | AI 자동 + 수정 |
| 색상 | 대표 색상 1~2개 | AI 자동 추출 |
| 계절감 | 봄/여름/가을/겨울/사계절 | 선택 |
| 스타일 태그 | 캐주얼, 포멀, 스포티 등 | AI 자동 + 수정 |
| 선호도 | ⭐ 1~5 | 선택 |
| 상태 | 활성/세탁중/보관중/처분 예정 | 선택 |

**옷장 관리 기능**

- 카테고리/색상/계절별 필터링 및 검색
- 옷장 통계 (카테고리 비율, 색상 분포, 가장 많이 입은 옷 등)
- "안 입는 옷" 알림 — 3개월 이상 미착용 의류 리마인드
- 계절 전환 시 옷장 정리 가이드

### 2-3. 🌤️ 날씨 API 연동 자동 추천

앱을 열면 날씨 기반으로 자동 추천이 이루어집니다.

**연동 데이터**

| 날씨 정보 | 활용 방식 |
|-----------|-----------|
| 현재 기온 / 체감 온도 | 의류 두께·레이어링 결정 |
| 최고/최저 기온 | 낮/밤 온도차 대비 아우터 추천 |
| 강수 확률 | 비 올 때 → 방수 아우터, 어두운 색 하의 추천 |
| 자외선 지수 | 높으면 → 모자, 긴팔 추천 |
| 습도 | 높으면 → 통풍 좋은 소재 추천 |
| 풍속 | 강풍 시 → 치마 대신 바지, 가벼운 아우터 주의 |

**날씨 기반 자동 동작**

- 앱 실행 시 위치 기반 날씨 자동 조회
- "오늘의 날씨 브리핑" 카드 (기온 + 옷차림 한 줄 가이드)
- 아침 푸시 알림: "오늘 12°C, 비 60% — 트렌치코트 + 어두운 바지 추천!"
- 주간 날씨 예보 기반 주간 코디 플래너 (프리미엄 기능)

**날씨 API 후보**

- OpenWeatherMap API (무료 티어 충분)
- 기상청 공공데이터 API (국내 정확도 높음)
- 두 API 병행 사용 권장 (해외 여행 시 OpenWeather, 국내는 기상청)

### 2-4. 💬 커뮤니티 / 공유 기능

다른 사용자의 코디를 참고하고 소통하는 공간입니다.

**핵심 기능**

- **오늘의 코디 공유** — 사진 + 착용 아이템 태그 + 상황 태그로 게시
- **코디 피드** — 인스타그램 스타일의 카드형 피드
- **좋아요 & 북마크** — 마음에 드는 코디 저장
- **스타일 태그 탐색** — #출근룩 #소개팅룩 #비오는날 등 태그 기반 탐색
- **코디 투표** — "오늘 A vs B 어떤 코디?" 투표 게시 기능
- **팔로우** — 취향이 비슷한 유저 팔로우
- **댓글** — 코디에 대한 피드백, 대안 제안

**커뮤니티 콘텐츠 구조**

```
게시글 = {
  사용자 사진 (전신 코디샷)
  + 착용 아이템 목록 (내 옷장에서 태그)
  + 상황 태그 (날씨, 장소, 대상)
  + 기분 이모지
  + 한 줄 코멘트
}
```

**탐색 방식**

- 추천 피드 (AI가 사용자 취향 기반 큐레이션)
- 인기 피드 (좋아요 순)
- 최신 피드 (시간순)
- 상황별 탐색 (출근, 데이트, 여행 등)
- 날씨별 탐색 (오늘과 비슷한 날씨의 과거 코디)

## 5. 로컬 개발 환경 설정

### 5-1. 빠른 시작 (API 키 없이 테스트)

```bash
# API 서버 실행 (SQLite + Mock 모드 자동 활성화)
cd apps/api && npm run start:dev

# 웹 앱 실행
cd apps/web && cp .env.local.example .env.local && npm run dev
```

- `USE_SQLITE=true` → Docker/PostgreSQL 없이 SQLite로 동작
- `SEED_DATA=true` → 앱 시작 시 테스트 유저 + 의류 20개 자동 생성
- API 키 미설정 시 Mock 모드 자동 전환:
  - AI 추천 → 규칙 기반 Mock
  - 날씨 → 서울 기반 계절별 Mock

**테스트 계정**

| 항목 | 값 |
|------|----|
| Email | test@moodfit.com |
| Password | test1234 |

### 5-2. 의류 데이터 수집 (무신사 크롤러)

```bash
# 전체 카테고리 수집
python scripts/crawl_musinsa.py

# 특정 카테고리 50개만 dry-run 확인
```

수집 카테고리: `top`(상의) / `outer`(아우터) / `bottom`(바지) / `shoes`(신발) / `bag`(가방) / `accessory`(액세서리)

## 6. 화면 구성 (주요 화면)

### 6-1. 화면 목록

| 화면 | 설명 |
|------|------|
| **홈** | 오늘 날씨 브리핑 + AI 코디 추천 카드 2~3개 |
| **추천 요청** | 기분·상황·대상 입력 → AI 추천 결과 |
| **내 옷장** | 등록된 의류 그리드 뷰 + 필터 |
| **옷 등록** | 사진 촬영/업로드 → AI 자동 분류 → 수정·저장 |
| **코디 캘린더** | 월간 달력 + 일별 착용 기록 |
| **커뮤니티 피드** | 코디 게시글 카드형 피드 |
| **게시글 작성** | 코디 사진 + 아이템 태그 + 상황 태그 |
| **프로필** | 내 정보, 스타일 프로필, 통계 |
| **설정** | 알림, 위치, 계정 관리 |

### 6-2. 사용자 플로우 (핵심)

```
앱 실행
  → 위치 기반 날씨 자동 조회
  → 홈 화면: 날씨 브리핑 + 자동 추천 코디 표시
  → [추천이 마음에 들면] "이 코디로 결정" → 착용 기록 저장
  → [다른 추천 원하면] "다시 추천" or 상황 직접 입력
  → [커뮤니티] 다른 사람 코디 탐색 → 북마크 → 참고하여 코디 결정
```
